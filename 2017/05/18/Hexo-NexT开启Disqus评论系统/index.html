Hexo-NexT开启Disqus评论系统

    选择你注册账号的用途，这里我们选I want to install disqus on my site

    填写你网站的相关信息

    选择付费套餐，这里当然是选择免费的Basic了

    之后的就不用选择了，已经可以使用了

  2. 编辑主题配置文件
